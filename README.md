# Introducción

Este repo contiene los tests funcionales, tanto para el ejercicio web como el ejercicio Api solicitado por Falabella


# ¿Qué necesitamos para ejecutar estos tests?

1. Java JDK
2. Maven
3. Geckodriver
4. Firefox

# Como corremos los tests

- ```mvn install```
- ```mvn test```

# Stack utilizado

- Rest Assured
- Maven
- Cucumber
- Selenium
- Junit
- Page object Model
